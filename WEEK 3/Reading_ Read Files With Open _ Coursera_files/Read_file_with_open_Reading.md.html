## Using Python&apos;s Open Function
27
​
28
Python&apos;s `open` function is used to create a file object and access the data within a text file. It takes two primary parameters:
29
​
30
1. **File Path**: The file path consists of the filename and directory where the file is located.
31
​
32
2. **Mode**: The mode parameter specifies the purpose of opening the file, such as &#39;r&#39; for reading, &#39;w&#39; for writing, or &#39;a&#39; for appending.
33

34
```python
35
# Step 1: Open the file in read ('r') mode
36

45
```

#### Step 1: Open the file in read (&#39;r&#39;) mode
52
​
53
**open(&#39;file.txt&#39;, &#39;r&#39;):** This line opens a file named &#39;file.txt&#39; in read mode (&#39;r&#39;). It returns a file object, which is stored in the variable file. The &#39;r&#39; mode indicates that the file will be opened for reading.
54
#### Step 2: Read the file content
55
​
56
**content = file.read():** Here, the read() method is called on the file object, which reads the entire content of the file and stores it in the variable content. This step effectively loads the entire content of &#39;file.txt&#39; into memory.

58
​
59

61
​
62
**file.close():** Finally, this line explicitly closes the file using the close() method. Closing the file is important to release system resources and ensure that the file is properly closed after reading. Failing to close the file can lead to resource leaks.
